## A functional ATM project written in Java

This project implements a simple ATM (Automated Teller Machine) simulator that stores the user's bank account, account number, and password. With this ATM, users can:
* Make deposits
* Withdraw money
* View their account balance

---

## Installation

### Requirements

- Java 8 or later

### Run locally (command line)

1. Clone this repository (or fork it first if you plan to contribute):
   git clone https://github.com/bhuvanapathuri22/ATM-Machine-Java.git
2. Change into the project directory:
   cd ATM-Machine-Java
3. Compile the Java source files:
   javac *.java
4. Run the program:
   java ATM

Notes:
- If you prefer an IDE, open the project folder in Eclipse, IntelliJ IDEA, or NetBeans and run the `ATM` class.
- Make sure `ATM.java`, `Account.java`, and `OptionMenu.java` are in the same package/folder when compiling with `javac`.
